# **App Name**: Horizon Hub

## Core Features:

- Content Summarization: Uses AI to provide summaries of articles.
- Comprehensive Display: Displays all components in one page with navigation.
- Project Display: Display the Real Estate projects in an image grid.
- Testimonial Carousel: A carousel to display client testimonials
- Contact Form: A contact form where the user can write down his contact information and his inquiry

## Style Guidelines:

- Primary color: Dark blue (#192A56) to convey trust and stability.
- Background color: Very light blue (#E8F0FE) to create a clean and professional look.
- Accent color: Orange (#FFA500) for highlights and call-to-action buttons to draw attention.
- Body and headline font: 'Inter', a sans-serif font for a modern, machined, objective and neutral feel. This font is suitable for both headlines and body text.
- Use Font Awesome icons for services and contact information.
- Use a grid layout for the projects section to showcase images effectively.
- Subtle fade-in animations on scroll to enhance user experience.
